# Expense Tracker Frontend

A responsive web application built using React for managing personal income and expenses efficiently.

---

## 🚀 Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3
- Axios
- React Hooks

---

## 📌 Features

- Responsive UI
- Add income and expenses
- View transaction history
- Edit transactions
- Delete transactions
- Category-wise tracking
- Real-time updates using REST APIs

---

## 📂 Project Structure

src
├── components
├── pages
├── services
├── styles
├── App.js
└── index.js

---

## ⚙️ Backend Connection

Backend API:

```
http://localhost:8080/api/transactions
```

Data communication is handled using Axios.

---

## ▶️ Running the Project

### Clone Repository

```bash
git clone https://github.com/your-username/expense-tracker-frontend.git
```

### Move into Project

```bash
cd expense-tracker-frontend
```

### Install Dependencies

```bash
npm install
```

### Start Application

```bash
npm start
```

Application runs on:

```
http://localhost:3000
```
